### 🚀 Étape 3 : Méthodes de déploiement

#### <img src="https://img.shields.io/badge/Render-12100E?style=for-the-badge&logo=render&color=white" height="28" />
- Déploiement rapide : [![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://bitbucket.org/dr-djibi/public_dockerfile_menma)

#### <img src="https://img.shields.io/badge/Koyeb-000000?style=for-the-badge&logo=koyeb&logoColor=white" height="28" />
- Déploiement rapide : [![Deploy to Koyeb](https://www.koyeb.com/static/images/deploy/button.svg)](https://app.koyeb.com/deploy?type=git&name=menma-md&repository=https%3A%2F%2Fgithub.com%2FDr-Djibi%2FDeploie-menma-md&branch=main&builder=dockerfile&instance_type=free)

#### <img src="https://img.shields.io/badge/Panel-grey?style=for-the-badge&logo=windows-terminal&logoColor=white" height="28" />
- Téléchargez l'archive : [**Télécharger Menma-MD (ZIP)**](https://github.com/Dr-Djibi/Menma-MD/archive/refs/heads/main.zip)
- Extrayez l'archive et importez tous les fichiers sur votre Panel.
- Créez un fichier `.env` à la racine ou configurez vos variables d'environnement dans l'onglet **Startup** de votre Panel.
- Démarrez simplement la console en lançant le bot via `npm start` ou `node index.js` (le fichier `index.js` d'origine gère l'hébergement Koyeb, Render et Panel de façon unifiée).
